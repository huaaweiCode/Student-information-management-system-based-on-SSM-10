# 基于SSM的学生信息管理系统10

#### Description
基于SSM的学生信息管理系统，拥有三种角色，学生、教师、管理员

- 管理员 : 具有所有管理模块的操控权限
- 学生 : 仅具有学生信息管理模块的查询及添加信息的权限
- 教师 : 仅具有学生信息管理模块的所有权限,且在教师信息管理模块中只具有查询及添加信息的权限
